In the strings.xml files able to change the application colors. The first color is the application basic color (light grey), and the second color is the splash screen (when the application starting) background color. By default this two color is same, but you can change it to different colors. If you want to make a bigger change in the user interface, then you can request a quote from me.

Configure UI modes - #back to top

Animated mode

  • Step 1 - Modify logo_shape value to 0.
  • Step 2 - Modify background_mode value to 0.

Cirlce mode

  • Step 1 - Modify logo_shape value to 0.
  • Step 2 - Modify background_mode value to 1.
  • Step 3 - Replace the radio_logo.jpg file in the mipmap folder (main\res\mipmap-xhdpi) with your radio logo.

Square mode

  • Step 1 - Modify logo_shape value to 1.
  • Step 2 - Modify background_mode value to 1.
  • Step 3 - Replace the radio_logo.jpg file in the mipmap folder (main\res\mipmap-xhdpi) with your radio logo.

Configure AdMob - #back to top

settings.xml 
                                false
                                ca-app-pub-3940256099942544~3347511713
                                ca-app-pub-3940256099942544/6300978111

The application use admob api to advertisements. Only admob banner is available in it. For the admob banner activation, change the admob_enabled property to true. After that, login to AdMob website, and generate admob_app_id and admob_unit_id, and replace the default values in the settings.xml file. For the admob id's generation, check this video.
